# Book Reviews
Book Reviews is an application for posting and reading book reviews. You can create an account add/edit books in the book index, rate and write reviews of books, and view reviews written by other users. 

## Installation
1. Clone this repo
2. Run `bundle install`
3. Run `rails db:migrate`
4. Run `rails s`
5. Navigate to http://localhost:3000
